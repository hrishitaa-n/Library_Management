# Library Management System

A simple Library Management System implemented in C++ using Linked Lists.

## Features

* Add new books to the library
* Display all books
* Display books by type/category
* Issue a book by book name
* Delete issued books from the library

## Technologies Used

* C++
* Linked Lists
* Object-Oriented Programming

## How to Compile and Run

Compile:

```bash
g++ library.cpp -o Library_Management
```

Run:

```bash
./Library_Management
```

On Windows:

```bash
Library_Management.exe
```

## Project Structure

* `library.cpp` - Main source code containing all functions and linked list implementation.
